Pampero
===

Aplicación web de Shiny para visualizar y analizar de manera interactiva datos de viento.

Requisitos
===

```R
library(shiny) 
library(openair) 
library(gnumeric) 
library(googleVis)
```

Para conexión a base de datos:

```R
library(DBI)
library(RMySQL)
```

Ejecutar app desde R: 
===

```R
runGitHub("Pampero", "guzmanlopez")
```
